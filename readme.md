# vite master
viteの開発環境を構築する

## 起動
```
git clone https://github.com/ShunsukeNONOMURA/vite-master.git
```

## 主要ライブラリ
- yarn create vite で設定
    - typescript
    - jsx
    - vue router
    - pinia
    - vitest
    - cypress
    - eslint
    - prettier
